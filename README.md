# 🎵 CBTis 47 - Music Streaming Platform

## 🚀 Descripción
Aplicación para gestionar playlists de música y video con una estructura moderna basada en NoSQL.

## 🧠 ¿Por qué NoSQL?

Una playlist se maneja como un documento que contiene:

- 📦 Arrays dinámicos  
- 🧩 Subdocumentos  
- ⚡ Operaciones push / pull  

💡 Permite mayor flexibilidad y rendimiento.
